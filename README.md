# 🍽️ Recipe Hub

A responsive and visually appealing **Cooking Recipes Website** built using **HTML and CSS only** as part of a web development assignment.

---

## Project Objective

The goal of this project is to design a **multi-page recipe website** that is:

* Clean and user-friendly
* Visually attractive
* Fully responsive (mobile + desktop)
* Built without using JavaScript

---

## Features

### 🏠 Home Page

* Website title: *Recipe Hub*
* Navigation bar
* Introduction section
* Featured recipes with images

### 🍲 Recipe Pages

Each recipe page includes:

* Recipe title
* Dish image
* Ingredients list
* Step-by-step instructions
* Preparation time
* Difficulty level (Easy / Medium / Hard)

---

## 🎨 CSS Highlights

The project demonstrates strong use of CSS concepts:

* 🎨 **Colors & Gradients** (modern UI design)
* ✍️ **Typography** (custom fonts, spacing, styling)
* 📦 **Box Model** (margin, padding, borders)
* 📐 **Flexbox Layout** (cards & navigation)
* ✨ **Hover Effects & Transitions**
* 📱 **Responsive Design (Media Queries)**
* 🎞️ **Animations** (scrolling text marquee)

---

## 💡 Bonus Features

* 🔍 Search bar (UI only)
* 🧾 Card-based layout for recipes
* 🎞️ Smooth animations and transitions
